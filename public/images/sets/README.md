- `116.jpg` - Collection Set 116
- `117.jpg` - Collection Set 117
- `118.jpg` - Collection Set 118
- `119.jpg` - Collection Set 119
- `120.jpg` - Collection Set 120

- `123.jpg` - Collection Set 123

- `125.jpg` - Collection Set 125
- `126.jpg` - Collection Set 126
- `127.jpg` - Collection Set 127
- `128.jpg` - Collection Set 128
- `129.jpg` - Collection Set 129

**Supported formats:** JPG, PNG, WEBP

**Recommended size:** 800x800px or larger (square format works best)

After placing the images, they will be automatically available at `/images/sets/[filename]`
