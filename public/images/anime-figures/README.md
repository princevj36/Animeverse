# Anime Figures Images Directory

Place your 11 anime figures images in this directory with the following naming convention:

- `130.jpg` - Anime Figure 130
- `131.jpg` - Anime Figure 131
- `132.jpg` - Anime Figure 132
- `133.jpg` - Anime Figure 133
- `134.jpg` - Anime Figure 134
- `135.jpg` - Anime Figure 135
- `136.jpg` - Anime Figure 136
- `137.jpg` - Anime Figure 137
- `138.jpg` - Anime Figure 138
- `139.jpg` - Anime Figure 139
- `140.jpg` - Anime Figure 140

**Supported formats:** JPG, PNG, WEBP

**Recommended size:** 800x800px or larger (square format works best)

After placing the images, they will be automatically available at `/images/anime-figures/[filename]`
