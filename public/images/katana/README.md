# Katana Images Directory

Place your 5 katana images in this directory with the following naming convention:

- `1.jpg` - Tanjiro Nichirin Blade Katana
- `2.jpg` - Zoro Enma Katana
- `3.jpg` - Ichigo Zangetsu Katana
- `4.jpg` - Sasuke Kusanagi Katana
- `5.jpg` - Mugen Champloo Katana

**Supported formats:** JPG, PNG, WEBP

**Recommended size:** 800x800px or larger (square format works best)

After placing the images, they will be automatically available at `/images/katana/[filename]`
