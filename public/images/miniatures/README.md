# Miniature Images Directory

Place your 10 miniature images in this directory with the following naming convention:

- `41.jpg` - Naruto Uzumaki Miniature
- `42.jpg` - Gaara Miniature
- `43.jpg` - Hinata Hyuga Miniature
- `44.jpg` - Sasuke Uchiha Miniature
- `45.jpg` - Jiraiya Miniature
- `46.jpg` - Itachi Uchiha Miniature
- `47.jpg` - Gaara Variant Miniature
- `48.jpg` - Itachi Akatsuki Miniature
- `84.jpg` - Luffy Miniature
- `85.jpg` - Sabo Miniature

**Supported formats:** JPG, PNG, WEBP

**Recommended size:** 800x800px or larger (square format works best)

After placing the images, they will be automatically available at `/images/miniatures/[filename]`
